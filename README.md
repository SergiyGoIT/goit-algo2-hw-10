# goit-algo2-hw-10

Домашнє завдання 10 з алгоритмів: складання розкладу занять за допомогою жадібного алгоритму для задачі покриття множини.

- `schedule.py` реалізує клас `Teacher` і функцію `create_schedule(subjects, teachers)`.

- Алгоритм на кожному кроці обирає викладача, який покриває найбільшу кількість ще не охоплених предметів.
- Якщо кілька викладачів покривають однакову кількість предметів, обирається наймолодший.

## Запуск

```bash
python3 schedule.py
```
